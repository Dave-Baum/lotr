Controls:

= : add token to selected card
- : remove token from selected card
[ : move selected card to back
] : move selected card to front
> : advance selected card to next phase
< : rewind selected card to previous phase
backspace : discard selected card

Buttons:

Shuffle: shuffle the encounter deck
Refresh: add discards to deck and shuffle
Put Top: put selected card on top of deck
Put Top: put selected card on bottom of deck
Reveal: play top card, face up
Shadow: play top card, face down
